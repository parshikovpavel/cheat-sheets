# Go Scheduler

## –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è

*Go scheduler* –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å–ª–µ–¥—É—é—â—É—é —Ç–µ—Ä–º–∏–Ω–æ–ª–æ–≥–∏—é:

–ü—Ä–∏ –∑–∞–ø—É—Å–∫–µ *Go program*'—ã,  –Ω–∞ –∫–∞–∂–¥–æ–µ *virtual core*, –æ–±–Ω–∞—Ä—É–∂–µ–Ω–Ω–æ–µ –Ω–∞ *host machine*, –Ω–∞–∑–Ω–∞—á–∞–µ—Ç—Å—è *Logical Processor (P !!!!)*. –ï—Å–ª–∏ —É –≤–∞—Å –µ—Å—Ç—å *processor* —Å –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ *hardware thread*'s –Ω–∞ –æ–¥–Ω–æ *physical cores* (—Ç–µ—Ö–Ω–æ–ª–æ–≥–∏—è *hyper-threading*), –∫–∞–∂–¥—ã–π *hardware thread* –±—É–¥–µ—Ç –≤–∏–¥–µ–Ω –∫–∞–∫ –æ—Ç–¥–µ–ª—å–Ω–æ–µ *virtual core*.

–í –¥–∞–Ω–Ω–æ–º –ø—Ä–∏–º–µ—Ä–µ –¥–ª—è MacBook Pro, —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `runtime.NumCPU()` –±—É–¥–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `8`. –õ—é–±–∞—è *Go program*, –∫–æ—Ç–æ—Ä—É—é —è –∑–∞–ø—É—â—É –Ω–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ, –ø–æ–ª—É—á–∏—Ç *8 P*.

–ö–∞–∂–¥–æ–º—É *P* –Ω–∞–∑–Ω–∞—á–∞–µ—Ç—Å—è *OS Thread* (¬´*M*¬ª !!!!). –ë—É–∫–≤–∞ ¬´*–ú*¬ª –æ–∑–Ω–∞—á–∞–µ—Ç ¬´–º–∞—à–∏–Ω–∞¬ª. –≠—Ç–æ—Ç *Thread* –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É —É–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è *OS*, –∏ *OS* –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É –æ—Ç–≤–µ—á–∞–µ—Ç –∑–∞ —Ä–∞–∑–º–µ—â–µ–Ω–∏–µ *Thread*'–∞ –Ω–∞ *Core* –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∫–æ–≥–¥–∞ —è –∑–∞–ø—É—Å–∫–∞—é *Go program* –Ω–∞ —Å–≤–æ–µ–π *machine*, —É –º–µ–Ω—è –µ—Å—Ç—å 8 *thread*'–æ–≤, –¥–æ—Å—Ç—É–ø–Ω—ã—Ö –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –º–æ–µ–π —Ä–∞–±–æ—Ç—ã, –∫–∞–∂–¥—ã–π –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –≤ –æ—Ç–¥–µ–ª—å–Ω–æ—Å—Ç–∏ –ø—Ä–∏–∫—Ä–µ–ø–ª–µ–Ω –∫ *P*.

–í –∫–∞–∂–¥–æ–π *Go program*'–µ –∏–∑–Ω–∞—á–∞–ª—å–Ω–æ —Å–æ–∑–¥–∞–µ—Ç—Å—è *initial Goroutine* (‚Äú*G*‚Äù !!!!), –∫–æ—Ç–æ—Ä–∞—è –∏—Å–ø–æ–ª–Ω—è–µ—Ç *Go program*'—É. –ü–æ–¥–æ–±–Ω–æ —Ç–æ–º—É, –∫–∞–∫ *OS Thread*'—ã *context-switch*'–∞—Ç—Å—è —Å –∏ –Ω–∞ *Core*, *Goroutine*'s *context-switch*'–∞—Ç—Å—è —Å –∏ –Ω–∞ *M*.

–í *Go scheduler*'–µ –µ—Å—Ç—å –¥–≤–µ *run queue*'s (–æ—á–µ—Ä–µ–¥–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è) ‚Äì *Global Run Queue* (*GRQ*, –≥–ª–æ–±–∞–ª—å–Ω–∞—è –æ—á–µ—Ä–µ–¥—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è) –∏ *Local Run Queue* (*LRQ*, –ª–æ–∫–∞–ª—å–Ω–∞—è –æ—á–µ—Ä–µ–¥—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è). –ö–∞–∂–¥–æ–º—É *P* –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è *LRQ*, –∫–æ—Ç–æ—Ä–∞—è —É–ø—Ä–∞–≤–ª—è–µ—Ç *Goroutine*'s, –ø—Ä–∏–≤—è–∑–∞–Ω–Ω—ã–º–∏ –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –≤ *context*'–µ *P* (?—á—Ç–æ —ç—Ç–æ –∑–Ω–∞—á–∏—Ç?). –≠—Ç–∏ *Goroutine*'s –∏–∑–≤–ª–µ–∫–∞—é—Ç—Å—è –ø–æ –æ—á–µ—Ä–µ–¥–∏, —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å *context-switch* –Ω–∞ –∏ —Å *M*, –ø—Ä–∏–≤—è–∑–∞–Ω–Ω–æ–≥–æ –∫ —ç—Ç–æ–º—É *P*. *GRQ* –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω –¥–ª—è *Goroutine*'s, –∫–æ—Ç–æ—Ä—ã–µ –µ—â–µ –Ω–µ –±—ã–ª–∏ –ø—Ä–∏–≤—è–∑–∞–Ω—ã –∫ *P*. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –ø—Ä–æ—Ü–µ—Å—Å –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏—è *Goroutine*'s –∏–∑ *GRQ* –≤ *LRQ*.

–ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–æ LRQ (–ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å GRQ) —è–≤–ª—è–µ—Ç—Å—è —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ—Å—Ç—å –∏ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –º–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞–Ω–∏—è. –û—Ç—Å—É—Ç—Å—Ç–≤—É–µ—Ç –≥–ª–æ–±–∞–ª—å–Ω—ã–π –¥–ª—è –≤—Å–µ—Ö *P* ‚Äì *mutex* –∏ —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –Ω–∏–º –±–ª–æ–∫–∏—Ä–æ–≤–∫–∏ –ø—Ä–∏ *sheduling*'e *goroutine*.

## Cooperating Scheduler

–°–≤—è–∑–∞–Ω–æ —Å [cooperative multitasking](../OS.md#cooperative-multitasking) –∏ [preemptive multitasking](../OS.md#preemptive-multitasking)

–ó–∞–¥–∞—á–∞ *Go scheduler*'–∞ ‚Äî —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–∏—Ç—å *runnable goroutin*'—ã –ø–æ –º–Ω–æ–∂–µ—Å—Ç–≤—É *OS thread*'–æ–≤.

*OS scheduler* —è–≤–ª—è–µ—Ç—Å—è *preemptive scheduler*. –í—ã –Ω–µ –º–æ–∂–µ—Ç–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞—Ç—å, —á—Ç–æ *scheduler* –±—É–¥–µ—Ç –¥–µ–ª–∞—Ç—å –≤ –∫–∞–∫–æ–π-—Ç–æ –º–æ–º–µ–Ω—Ç –≤—Ä–µ–º–µ–Ω–∏. *OS kernel* (—è–¥—Ä–æ –û–°) –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Ä–µ—à–µ–Ω–∏—è –∏ –æ–Ω–∏ –Ω–µ–¥–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω—ã. –ü—Ä–∏–ª–æ–∂–µ–Ω–∏—è, —Ä–∞–±–æ—Ç–∞—é—â–∏–µ –ø–æ–≤–µ—Ä—Ö *OS*, –Ω–µ –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É—é—Ç —Ç–æ, –∫–∞–∫ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è *sheduling* –≤–Ω—É—Ç—Ä–∏ *kernel*. –î–ª—è —Ç–∞–∫–æ–≥–æ –∫–æ–Ω—Ç—Ä–æ–ª—è –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ *synchronization primitive*'s, —Ç–∞–∫–∏–µ –∫–∞–∫ *atomic* –∏ *mutex*.

*Go scheduler* —è–≤–ª—è–µ—Ç—Å—è —á–∞—Å—Ç—å—é *Go runtime*, –∞ *Go runtime*, –≤ —Å–≤–æ—é –æ—á–µ—Ä–µ–¥—å, –≤—Å—Ç—Ä–æ–µ–Ω –≤ –≤–∞—à–µ *application*. –ü–æ—ç—Ç–æ–º—É *Go scheduler* —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ [*user space*](../OS.md#user-space) –Ω–∞–¥ *kernel*. –ù–∞ —Å–µ–≥–æ–¥–Ω—è, *Go scheduler* —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω –∫–∞–∫ *cooperative* (*non-preemptive*) *multitasking*. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ *scheduler*'—É –Ω—É–∂–Ω—ã —á–µ—Ç–∫–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ *user space event*'—ã, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –≤ *safe point*'–∞—Ö –≤ –∫–æ–¥–µ, —á—Ç–æ–±—ã –ø—Ä–∏–Ω–∏–º–∞—Ç—å —Ä–µ—à–µ–Ω–∏—è –æ *scheduling*'–µ.

–ü—Ä–∏ —ç—Ç–æ–º *Go cooperating scheduler* –≤—ã–≥–ª—è–¥–∏—Ç –∏ –æ—â—É—â–∞–µ—Ç—Å—è –∫–∞–∫ *preemptive*. –í—ã –Ω–µ –º–æ–∂–µ—Ç–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞—Ç—å, —á—Ç–æ —Å–æ–±–∏—Ä–∞–µ—Ç—Å—è –¥–µ–ª–∞—Ç—å *Go scheduler*. –≠—Ç–æ —Å–≤—è–∑–∞–Ω–æ —Å —Ç–µ–º, —á—Ç–æ –ø—Ä–∏–Ω—è—Ç–∏–µ —Ä–µ—à–µ–Ω–∏–π –¥–ª—è *Go cooperating scheduler*'–∞ –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è *Go runtime*, –∞ –Ω–µ —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–æ–º. –ü–æ—ç—Ç–æ–º—É –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≤–æ—Å–ø—Ä–∏–Ω–∏–º–∞—Ç—å *Go scheduler* –∫–∞–∫ *preemptive scheduler*, –ø–æ—Å–∫–æ–ª—å–∫—É *scheduler* –Ω–µ–¥–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω.

## *Goroutine State*'s

–ö–∞–∫ –∏ *Thread*'—ã (–ø–æ–¥—Ä–æ–±–Ω–µ–µ –ø—Ä–æ [thread state](../OS.md#thread-state)), *Goroutine*'s –∏–º–µ—é—Ç —Ç–µ –∂–µ —Ç—Ä–∏ –≤—ã—Å–æ–∫–æ—É—Ä–æ–≤–Ω–µ–≤—ã—Ö *state*'s. –≠—Ç–∏ *state* –≤–ª–∏—è—é—Ç –Ω–∞ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ *Go scheduler*'–∞ —ç—Ç–æ–π *goroutine*.  *Goroutine* –º–æ–∂–µ—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤ –æ–¥–Ω–æ–º –∏–∑ —Ç—Ä–µ—Ö *state*'s:

- `waiting` (–æ–∂–∏–¥–∞—é—â–∏–π) ‚Äì —ç—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ *goroutine* –æ—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∞ –∏ –æ–∂–∏–¥–∞–µ—Ç (*waiting*) —á–µ–≥–æ-—Ç–æ, —á—Ç–æ–±—ã –ø—Ä–æ–¥–æ–ª–∂–∏—Ç—å —Ä–∞–±–æ—Ç—É. –≠—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å *waiting* OS (*system call*) –∏–ª–∏ *synchronization call* (*atomic, mutex*). –≠—Ç–∏ —Ç–∏–ø—ã [latency](https://en.wikipedia.org/wiki/Latency_(engineering)) —è–≤–ª—è—é—Ç—Å—è *root cause* –Ω–∏–∑–∫–æ–≥–æ *performance*.
- `runnable` (–≥–æ—Ç–æ–≤—ã–π –∫ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—é) ‚Äì –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ *goroutine* –Ω—É–∂–Ω–æ *time* –Ω–∞ *M*, —á—Ç–æ–±—ã –æ–Ω–∞ –º–æ–≥–ª–∞ –≤—ã–ø–æ–ª–Ω—è—Ç—å —Å–≤–æ–∏ *instruction*'s. –ü—Ä–∏ –±–æ–ª—å—à–æ–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ *goroutine*'s –≤ —Ç–∞–∫–æ–º —Å–æ—Å—Ç–æ—è–Ω–∏–∏, –æ–Ω–∏ –±—É–¥—É—Ç –≤—ã–Ω—É–∂–¥–µ–Ω—ã –¥–æ–ª–≥–æ –æ–∂–∏–¥–∞—Ç—å –¥–æ –º–æ–º–µ–Ω—Ç–∞ –ø–æ–ª—É—á–µ–Ω–∏—è *time*. –ö—Ä–æ–º–µ —Ç–æ–≥–æ, –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ *time*, –∫–æ—Ç–æ—Ä–æ–µ –±—É–¥–µ—Ç –ø–æ–ª—É—á–∞—Ç—å –∫–∞–∂–¥–∞—è *goroutine*, –±—É–¥–µ—Ç —Å–æ–∫—Ä–∞—â–∞—Ç—å—Å—è –ø–æ –º–µ—Ä–µ —Ç–æ–≥–æ, –∫–∞–∫ –≤—Å–µ –±–æ–ª—å—à–µ *goroutine*'s –±—É–¥–µ—Ç —Å–æ—Ä–µ–≤–Ω–æ–≤–∞—Ç—å—Å—è –∑–∞ –≤—Ä–µ–º—è. –≠—Ç–æ—Ç —Ç–∏–ø *scheduling latency* —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–∏—á–∏–Ω–æ–π –Ω–∏–∑–∫–æ–≥–æ *performance*.
- `executing` (–∏—Å–ø–æ–ª–Ω—è—é—â–∏–π—Å—è) ‚Äì –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ *goroutine* –±—ã–ª–∞ –ø–æ–º–µ—â–µ–Ω–∞ –Ω–∞ *M* –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Å–≤–æ–∏ *instruction*'s. 



## Context Switching

*Go scheduler*'—É —Ç—Ä–µ–±—É—é—Ç—Å—è —á–µ—Ç–∫–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ *user-space event*'—ã, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –≤ *safe point*'–∞—Ö –≤ –∫–æ–¥–µ, —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å *context switch*. –≠—Ç–∏ *event*'—ã –∏ *safe point*'—ã –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ –º–µ—Å—Ç–∞—Ö *function call*'s. *Function call*'s –∏–º–µ—é—Ç –∫—Ä–∏—Ç–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è —Ä–∞–±–æ—Ç–æ—Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç–∏ *Go scheduler*'–∞. –î–ª—è Go 1.11 –∏–ª–∏ –Ω–∏–∂–µ, –µ—Å–ª–∏ –≤—ã –∑–∞–ø—É—Å–∫–∞–µ—Ç–µ –∫–∞–∫–∏–µ-–ª–∏–±–æ –¥–æ–ª–≥–∏–µ —Ü–∏–∫–ª—ã, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –¥–µ–ª–∞—é—Ç *function call*'s, —ç—Ç–æ –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ –∑–∞–¥–µ—Ä–∂–∫–∞–º *Go scheduler*'–∞ –∏ GC. –ö—Ä–∏—Ç–∏—á–µ—Å–∫–∏ –≤–∞–∂–Ω–æ, —á—Ç–æ–±—ã *function call*'s –ø—Ä–æ–∏—Å—Ö–æ–¥–∏–ª–∏ –≤ —Ä–∞–∑—É–º–Ω—ã–µ —Å—Ä–æ–∫–∏.

–ü—Ä–µ–¥–ª–æ–∂–µ–Ω *[proposal](https://github.com/golang/go/issues/24543)* –¥–ª—è Go 1.12, –∫–æ—Ç–æ—Ä–æ–µ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ *preemptive* (*non-cooperative*) —Ç–µ—Ö–Ω–∏–∫ –≤–Ω—É—Ç—Ä–∏ *Go scheduler*'–∞, —á—Ç–æ–±—ã –æ–±–µ—Å–ø–µ—á–∏—Ç—å *preemption* (–≤—ã—Ç–µ—Å–Ω–µ–Ω–∏–µ) –¥–æ–ª–≥–∏—Ö —Ü–∏–∫–ª–æ–≤.

–ï—Å—Ç—å —á–µ—Ç—ã—Ä–µ –∫–ª–∞—Å—Å–∞ *event*'–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–∑–≤–æ–ª—è—é—Ç *scheduler*'—É –ø—Ä–∏–Ω–∏–º–∞—Ç—å —Ä–µ—à–µ–Ω–∏—è –æ *scheduling*'–µ: 

- –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ `go` *statement* ‚Äì –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è *goroutine*'s. 

  –ö–∞–∫ —Ç–æ–ª—å–∫–æ –Ω–æ–≤–∞—è *goroutine* —Å–æ–∑–¥–∞–Ω–∞, –æ–Ω–∞ –¥–∞–µ—Ç *scheduler*'—É –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –ø—Ä–∏–Ω—è—Ç—å —Ä–µ—à–µ–Ω–∏–µ –æ *scheduling*'–µ.

- *Garbage collection*

  –ü–æ—Å–∫–æ–ª—å–∫—É GC —Ä–∞–±–æ—Ç–∞–µ—Ç —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Å–æ–±—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ –Ω–∞–±–æ—Ä–∞ *goroutine*'s, —ç—Ç–∏ *goroutine*'s —Ç—Ä–µ–±—É—é—Ç –≤—Ä–µ–º—è –Ω–∞ M –¥–ª—è –∑–∞–ø—É—Å–∫–∞. –≠—Ç–æ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ —Ç–æ–º—É, —á—Ç–æ GC —Å–æ–∑–¥–∞–µ—Ç —Ö–∞–æ—Å –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ *scheduling*'–∞. –û–¥–Ω–∞–∫–æ *scheduler* –ø–æ–Ω–∏–º–∞–µ—Ç, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–∞–∂–¥–∞—è *goroutine* –∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —ç—Ç–æ –∑–Ω–∞–Ω–∏–µ –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ *scheduling*'–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–µ–ª–∞–µ—Ç *context-switching* —Å *goroutine*, –∫–æ—Ç–æ—Ä–∞—è –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç —Å *heap*  –≤–æ –≤—Ä–µ–º—è GC, –Ω–∞ *goroutine*, –∫–æ—Ç–æ—Ä–∞—è –Ω–µ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç —Å *heap* –≤ —ç—Ç–æ –≤—Ä–µ–º—è. –ö–æ–≥–¥–∞ GC —Ä–∞–±–æ—Ç–∞–µ—Ç, –ø—Ä–∏–Ω–∏–º–∞–µ—Ç—Å—è –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ä–µ—à–µ–Ω–∏–π –ø–æ *scheduling*'—É.

- *System call*'s

  –ï—Å–ª–∏ *goroutine* –¥–µ–ª–∞–µ—Ç *system call*, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ –±–ª–æ–∫–∏—Ä–æ–≤–∫–µ *–ú*, *scheduler* –º–æ–∂–µ—Ç *context-switch off* —ç—Ç—É *goroutine* —Å –ú –∏ *context-switch on* –Ω–æ–≤—É—é *goroutine* –Ω–∞ —Ç—É –∂–µ —Å–∞–º—É—é *–ú*. –ù–æ –∏–Ω–æ–≥–¥–∞ —Ç–∞–∫–æ–π *context-switch* –Ω–µ–≤–æ–∑–º–æ–∂–µ–Ω, –∏ —Ç—Ä–µ–±—É–µ—Ç—Å—è –Ω–æ–≤–∞—è –ú –¥–ª—è –ø—Ä–æ–¥–æ–ª–∂–µ–Ω–∏—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è *goroutine*'s, –∫–æ—Ç–æ—Ä—ã–µ —Å—Ç–æ—è—Ç –≤ –æ—á–µ—Ä–µ–¥–∏ –∫ P. –î–∞–ª–µ–µ —ç—Ç–æ –æ–±—ä—è—Å–Ω—è–µ—Ç—Å—è –ø–æ–¥—Ä–æ–±–Ω–µ–µ.

- *Synchronization* –∏ *orchestration*

  –ï—Å–ª–∏ –≤—ã–∑–æ–≤ –æ–ø–µ—Ä–∞—Ü–∏–π ‚Äì *atomic*, *mutex* –∏–ª–∏ *channel* ‚Äì –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ –±–ª–æ–∫–∏—Ä–æ–≤–∫–µ *goroutine*, *scheduler* –º–æ–∂–µ—Ç —Å–¥–µ–ª–∞—Ç—å *context-switch* –Ω–∞ –Ω–æ–≤—É—é *goroutine*. –ö–∞–∫ —Ç–æ–ª—å–∫–æ *goroutine* —Å–º–æ–∂–µ—Ç —Å–Ω–æ–≤–∞ –∑–∞–ø—É—Å—Ç–∏—Ç—å—Å—è, –µ–µ –º–æ–∂–Ω–æ –ø–æ–≤—Ç–æ—Ä–Ω–æ –ø–æ—Å—Ç–∞–≤–∏—Ç—å –≤ –æ—á–µ—Ä–µ–¥—å –∏, —á–µ—Ä–µ–∑ –∫–∞–∫–æ–µ-—Ç–æ –≤—Ä–µ–º—è, –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç *context-switch* –Ω–∞ –Ω–µ–µ –æ–±—Ä–∞—Ç–Ω–æ.

–≠—Ç–æ –Ω–µ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ *scheduling* –±—É–¥–µ—Ç –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç—å –∫–∞–∂–¥—ã–π —Ä–∞–∑ –ø—Ä–∏ –Ω–∞—Å—Ç—É–ø–ª–µ–Ω–∏–∏ –∫–∞–∂–¥–æ–≥–æ –∏–∑ —ç—Ç–∏—Ö —Å–æ–±—ã—Ç–∏–π. –≠—Ç–æ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ *scheduler* –ø–æ–ª—É—á–∏—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å *scheduling*'–∞ –ø—Ä–∏ –∫–∞–∂–¥–æ–º –∏–∑ —ç—Ç–∏—Ö *event*'–æ–≤.

## –ê—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã–µ *system call*'s

–ö–æ–≥–¥–∞ OS —É–º–µ–µ—Ç –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å *system call*, –¥–ª—è –±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–π –æ–±—Ä–∞–±–æ—Ç–∫–∏ *system call* –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —á—Ç–æ-—Ç–æ –≤—Ä–æ–¥–µ [network poller](https://golang.org/src/runtime/netpoll.go). –≠—Ç–æ –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é *kqueue* (MacOS), *epoll* (Linux) –∏–ª–∏ *iocp* (Windows) –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏—Ö –û–°.

*Network* *system call*'s –º–æ–≥—É—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å—Å—è –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ –º–Ω–æ–≥–∏–º–∏ –û–°, –∫–æ—Ç–æ—Ä—ã–µ –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º —Å–µ–≥–æ–¥–Ω—è. –ò–º–µ–Ω–Ω–æ –æ—Ç—Å—é–¥–∞ *network poller* –ø–æ–ª—É—á–∏–ª —Å–≤–æ–µ –Ω–∞–∑–≤–∞–Ω–∏–µ, –ø–æ—Å–∫–æ–ª—å–∫—É –µ–≥–æ –æ—Å–Ω–æ–≤–Ω–æ–µ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏–µ ‚Äî –æ–±—Ä–∞–±–æ—Ç–∫–∞ *network* –æ–ø–µ—Ä–∞—Ü–∏–π. –ò—Å–ø–æ–ª—å–∑—É—è *network poller* –¥–ª—è *networking system call*'s, *scheduler* –º–æ–∂–µ—Ç –Ω–µ –ø–æ–∑–≤–æ–ª—è—Ç—å *Goroutin*'–∞–º –±–ª–æ–∫–∏—Ä–æ–≤–∞—Ç—å **M** –ø—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ *system call*'s. –≠—Ç–æ –ø–æ–º–æ–≥–∞–µ—Ç —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å –¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç—å **M** –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –¥—Ä—É–≥–∏—Ö *Goroutine*'s –∏–∑ **LRQ** (–ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–∞ **P**) –±–µ–∑ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ —Å–æ–∑–¥–∞–Ω–∏—è –Ω–æ–≤–æ–π **M**. –≠—Ç–æ –ø–æ–º–æ–≥–∞–µ—Ç —Å–Ω–∏–∑–∏—Ç—å –Ω–∞–≥—Ä—É–∑–∫—É –Ω–∞ –û–° –ø–æ *scheduling*'—É.

## Concurrency

*Concurrency* –æ–∑–Ω–∞—á–∞–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ ¬´–Ω–µ –ø–æ –ø–æ—Ä—è–¥–∫—É¬ª. –í–∑—è—Ç—å –Ω–∞–±–æ—Ä *instruction*'s, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –∏ –Ω–∞–π—Ç–∏ —Å–ø–æ—Å–æ–± –≤—ã–ø–æ–ª–Ω–∏—Ç—å –∏—Ö "–Ω–µ –ø–æ –ø–æ—Ä—è–¥–∫—É", –Ω–æ –ø—Ä–∏ —ç—Ç–æ–º –ø–æ–ª—É—á–∏—Ç—å —Ç–æ—Ç –∂–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç. –ü—Ä–∏—á–µ–º —ç—Ç–æ –¥–æ–ª–∂–Ω–æ –∏–º–µ—Ç—å —Ü–µ–Ω–Ω–æ—Å—Ç—å ‚Äì –¥–æ—Å—Ç–∞—Ç–æ—á–Ω—ã–π —Ä–æ—Å—Ç *performance* –∫ –∑–∞—Ç—Ä–∞—Ç–∞–º –Ω–∞ —Å–ª–æ–∂–Ω–æ—Å—Ç—å. –í –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –ø—Ä–æ–±–ª–µ–º—ã, –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ "–Ω–µ –ø–æ –ø–æ—Ä—è–¥–∫—É" –º–æ–∂–µ—Ç –æ–∫–∞–∑–∞—Ç—å—Å—è –Ω–µ–≤–æ–∑–º–æ–∂–Ω—ã–º –∏–ª–∏ –¥–∞–∂–µ –Ω–µ –∏–º–µ—Ç—å —Å–º—ã—Å–ª–∞.

[Concurrency ‚Äì —ç—Ç–æ –Ω–µ —Ç–æ–∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ parallelism](https://blog.golang.org/concurrency-is-not-parallelism). *Parallelism* –æ–∑–Ω–∞—á–∞–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –¥–≤—É—Ö –∏–ª–∏ –±–æ–ª–µ–µ *instruction*'s –≤ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –≤—Ä–µ–º—è. *Parallelism* –≤–æ–∑–º–æ–∂–µ–Ω —Ç–æ–ª—å–∫–æ –≤ —Ç–æ–º —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ —É –≤–∞—Å –µ—Å—Ç—å –∫–∞–∫ –º–∏–Ω–∏–º—É–º –¥–≤–∞ *OS thread*'–∞,  –¥–≤–∞ *Core*'s –∏ –¥–≤–µ *Goroutine*'s, –∫–∞–∂–¥–∞—è –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –≤—ã–ø–æ–ª–Ω—è–µ—Ç *instruction*'s –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –≤ –æ—Ç–¥–µ–ª—å–Ω–æ–º *OS thread*'–µ.

![img](../img/go/96_figure1.png)

–ù–∞ —Ä–∏—Å—É–Ω–∫–µ ‚Äì –¥–≤—É—Ö *processor*'–∞ (**P**), –∫ –∫–∞–∂–¥–æ–º—É –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –ø—Ä–∏–∫—Ä–µ–ø–ª–µ–Ω –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã–π *OS thread* (**M**), —Å–≤—è–∑–∞–Ω–Ω—ã–π —Å –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã–º *hardware thread*'–æ–º (**Core**) (–Ω—É –Ω–∞–≤–µ—Ä–Ω–æ–µ —ç—Ç–æ –Ω–µ –ø—Ä—è–º *thread*). –î–≤–µ *Goroutine*'s (G1 –∏ G2) –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –≤ *parallel*, –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –≤—ã–ø–æ–ª–Ω—è—è —Å–≤–æ–∏ *instruction*'s –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–º *OS/hardware thread*'–µ. –ö –∫–∞–∂–¥–æ–º—É *processor*'—É –ø—Ä–∏–≤—è–∑–∞–Ω—ã –ø–æ —Ç—Ä–∏ *Goroutine*'s (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∫ **P1** ‚Äì **G2**, **G3**, **G5**), –∫–æ—Ç–æ—Ä—ã–µ –ø–æ –æ—á–µ—Ä–µ–¥–∏ —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π *OS thread*. –≠—Ç–∏ *Goroutin*'—ã —Ä–∞–±–æ—Ç–∞—é—Ç *concurrently*, –æ–Ω–∏ *share* –≤—Ä–µ–º—è –Ω–∞ *OS thread*'–µ –∏ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –±–µ–∑ –∑–∞—Ä–∞–Ω–µ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–≥–æ –ø–æ—Ä—è–¥–∫–∞.

## –í–∏–¥—ã —Ä–∞–±–æ—á–µ–π –Ω–∞–≥—Ä—É–∑–∫–∏ (workload)

–¢–∏–ø *workload*, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç *goroutine*, –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ–Ω—è—Ç—å ‚Äì –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ª–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å *parallelism* –¥–ª—è –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–∏ *concurrency*. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ *workload*'s:

# Alignment / padding

*Data structure alignment* (–≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö) - —Å–ø–æ—Å–æ–± —Ä–∞–∑–º–µ—â–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –≤ –ø–∞–º—è—Ç–∏ –æ—Å–æ–±—ã–º –æ–±—Ä–∞–∑–æ–º –¥–ª—è —É—Å–∫–æ—Ä–µ–Ω–∏—è –¥–æ—Å—Ç—É–ø–∞.. –û–Ω–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ç—Ä–µ—Ö –æ—Ç–¥–µ–ª—å–Ω—ã—Ö, –Ω–æ –≤–∑–∞–∏–º–æ—Å–≤—è–∑–∞–Ω–Ω—ã—Ö –ø—Ä–æ–±–ª–µ–º: *data alignment* (–≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö), *data structure padding* (–æ—Ç—Å—Ç—É–ø —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö) –∏ *packing* (—É–ø–∞–∫–æ–≤–∫–∞).
